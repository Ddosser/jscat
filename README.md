基于Windows Script Host的简单RAT！想法来源于Koadic和JSRat-Py，并不是要做一款功能全面的强大的远控，最终目标是轻量、流量加密、免杀和CS加载器。



### Features

- 反弹上线/自动上线/无文件落地
- 流量加密(rc4)
- 命令执行
- 上传/下载
- shellcode注入(DotNetToJS)

### Install

```
# git clone https://github.com/hanc00l/jscat
# cd jscat
# pip3 install -r requirements.txt
```
